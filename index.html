About these tools

Each calculator is a single, self-contained HTML file with no build step or server required. The calculators can be used:

  • Directly in a browser — open the file or serve as a static site
  • Embedded in GitEHR as a Tauri webview window, with results saved as structured patient record entries
  • Embedded in any web application via <iframe> with postMessage result handoff

Scoring logic is implemented in plain JavaScript, validated against the authoritative clinical references in clinical-source-references/.
